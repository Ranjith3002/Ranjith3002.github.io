3.list all global attributes in html

  • **id**: Unique identifier for an element within a document.
  • **class**: One or more space-separated values that define the class names of an element.
  • **style**: Inline style declarations for an element.
  • **title**: Provides a tooltip for an element.
  • **lang**: Specifies the language of the element's content.
  • **dir**: Specifies the direction of the text in an element.
  • **hidden**: Hides an element from the user.
  • **tabindex**: Specifies the order in which an element is focused when navigating with the tab key.
  • **accesskey**: Specifies a shortcut key for an element.
  • **contenteditable**: Indicates whether the content of an element can be edited by the user.
  • **spellcheck**: Indicates whether the content of an element should be spell-checked.
  • **data-* attributes**: Custom attributes that can be used to store arbitrary data on an element.
